# React Dashboard Project

## Description
A modern and responsive admin dashboard built with **React** and **Vite**.  
It features **charts, tables, and navigation** and can be used as a base template for admin panels or analytics apps.

## Features
- Responsive sidebar and top navigation  
- Charts and statistics display  
- User tables with search & pagination  
- Dark/Light mode toggle  
- Easy to customize  

## Technologies
HTML, Tailwind CSS, React, Vite, React Router DOM  

## Setup
```bash
npm install
npm run dev
